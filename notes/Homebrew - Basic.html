Homebrew - Basic

Created: January 7, 2022 11:11 AM
Last Edited Time: February 3, 2024 2:44 PM
Tags: mac

Installation

brew install [package-name]

Removal

brew uninstall [package-name]

Update

brew update  # Update Homebrew itself
brew upgrade [package-name]  # Upgrade a specific package
brew upgrade  # Upgrade all packages

Clean cache

brew cleanup

List installed packages

brew list

Search packages

  • Search for packages matching a keyword
brew search [text]

Check information

  • Get information about a specific package
brew info [package]
